获广西计算机应用大赛三等奖作品

由于本网站采用PHP全动态网页设计，所以首页文件不使用index.htm。

运行环境：PHP，MySQL(PHP建议版本：5.2及以上)

注意：此网页为动态网页，依赖于PHP环境，和MySQL数据库。
本地演示：可以下载phpstudy 一键安装，将本网站目录放至phpstudy网站的根目录，配置数据库密码，导入数据即可运行。

数据导入：运行 /chuxin/page/setup/index.php 即可一键导入

数据库配置文件：修改page文件夹的config文件
